搞清楚夏普指数计算#

参考了：https://www.learnpythonwithrune.org/how-to-calculate-sharpe-ratio-with-pandas-and-numpy/

import pandas as pd
import numpy as np
import altair as alt
import akshare as ak

df = ak.stock_zh_a_hist(symbol="603777", period="daily", adjust="hfq")
test = df.iloc[:,[0,2]]
test.columns = ['td', 'close']

# 查看原始数据大约情况
# altair中，需要指定x轴为时间类型，才能缩放x轴。
alt.Chart(test).mark_line().encode(
    x = 'td:T',
    y = 'close:Q',
).properties(width = 600).interactive()

test.set_index('td', inplace=True)
log_rt = np.log(test/test.shift())

# 查看一下日回报的分布
alt.Chart(log_rt).mark_bar().encode(
    alt.X('close:Q', bin=True),
    y= 'count()',
)

# 计算夏普指数并年化
s_ratio = log_rt.mean()/log_rt.std()
s_ratio*252**.5

close    0.103558
dtype: float64

上面计算年化的公式#

  • 年化指数回报=log_return.mean()*252

  • 年化波动率=std*252^.5

  • 两个相除得到：anual_ratio = day_ratio*252^0.5

# 另外一种方法是计算几何平均收益率
d2y = test.shape[0] / 252 # 计算出年数
whole_rt = test.tail(1).iloc[0,0] / test.head(1).iloc[0,0] # 总回报=最后一天除以第一天
alg_rt = whole_rt**(1/d2y)-1 #几何平均
# 波动率用pct_change
alg_std = test.pct_change().std()*(252**0.5)

alg_rt / alg_std

close    0.105845
dtype: float64

总结#

  • 搞清楚夏普比率的准确意思和算法。

  • 使用指数收益率和几何平均收益率，最终结果差别不大。

  • 波动率年化的公式还是不太理解，有个解释是：平方根的原因是假设收益是正态随即过程。

  • 用收益率的缺点很明显的，和起终点关系太大了。考虑用贝塔值可能会好一点？

def sharpe_logr(s):
    log_rt = np.log(s/s.shift())
    sharpe = log_rt.mean() / log_rt.std()
    anual_ratio = sharpe*252**0.5

    return anual_ratio

sharpe_logr(test.iloc[:,0])
